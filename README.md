# 🚀 Finsight Pro – AI-Powered Personal Finance Dashboard

Finsight Pro is a full-stack personal finance web application that helps users track expenses, manage budgets, analyze spending behavior, and receive AI-driven financial insights — all in a clean, modern dashboard.

---

## ✨ Key Features

- 📊 **Expense Tracking & Analytics**
  - Add, edit, filter, and categorize expenses
  - Visual insights via charts (category & monthly trends)

- 💡 **AI Financial Advisor**
  - Personalized suggestions based on user spending patterns
  - Detects overspending and recommends optimizations

- ⚠️ **Anomaly Detection**
  - Identifies unusual spending spikes across categories
  - Highlights behavioral changes month-over-month

- 📈 **Budget Forecasting**
  - Predicts future spending using historical data
  - Warns when users may exceed their monthly budget

- 🤖 **AI Copilot Chatbot**
  - Finance-focused assistant for user queries
  - Provides insights like “Where am I overspending?” or “How can I save more?”

- 🧾 **Receipt Scanner (Manual + AI-ready)**
  - Upload receipts and extract expense data (extensible for OCR/AI parsing)

- 🏦 **Finance & EMI Management**
  - Track salary, budget, and savings
  - Monitor EMI burden and risk level

- 🎯 **Demo Mode + Onboarding**
  - Interactive onboarding for new users
  - Safe demo dataset for exploration without affecting real data

- REST API architecture

### Database
